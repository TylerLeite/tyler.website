The automatic mosaic thing

This is a program that takes an image and creates a mosaic out of it. It reads in pixel data and makes an educated guess about which color that pixel is. It then uses that information to construct an HTML table where each cell is representative of a pixel. All photos are either taken by me or of me, so hopefully no one gets mad about copyright. As of right now, you can't make a mosaic from an arbitrary image's url (for security reasons) or upload images (for github.io reasons), so hopefully my beautiful photography is good enough for you.
