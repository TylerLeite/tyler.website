So here, I guess I'll give you a sneak peak at something I've been tinkering with.

It's a random text generator. Basically, it takes a bunch of phonemes (the building blocks of words) and puts them together mostly randomly. By this method, it builds up a vocabulary and stores it in a "dictionary". It then picks words randomly from this dictionary and separates them with periods (full-stops) and commas (probably half-stops, maybe a little less). Behind the scenes, there are a bunch of variables that control things like average syllable count and lexical density. These variables make up a genome of sorts. The code that I have written so far is a set of instructions for how to express that genome. Big picture, this is going to be part of a project where I analyze text from a bunch of different languages to create a "genotype pool" and then make hybrid and imitation languages like some kind of mad scientist.
